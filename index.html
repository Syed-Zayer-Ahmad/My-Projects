⚡ Energy Duel

✓ Complete

📖 Description

A strategic two-player battle game where players manage energy points to execute powerful attacks. The goal is to reduce your opponent's health to zero through smart tactical decisions.

🎮 Game Mechanics

  • Light Attack: Costs 1 EP, deals 10 damage
  • Heavy Attack: Costs 3 EP, deals 35 damage
  • Charge: Gains 2 energy points (free action)
  • Both players start with 100 HP and 0 EP
💡 Strategy Tip: Balance between attacking and charging energy to set up powerful combos!

🔥 Element Duel

✓ Complete

📖 Description

An advanced two-player strategy game featuring three elemental stances. Read your opponent's moves and choose the winning stance to maximize damage and healing!

🎮 Game Mechanics

  • Fire Stance: High offensive damage
  • Water Stance: Healing and tactical defense
  • Earth Stance: Protective and resilient
  • Dynamic damage scaling based on round progression
💡 Difficulty Levels: Rounds 1-3 (1x), Rounds 4-6 (1.5x), Rounds 7+ (2x - Sudden Death!)

🚀 How to Run the Games

Prerequisites

Make sure you have a C compiler installed (gcc, clang, or MSVC):

1️⃣ Energy Duel

gcc example_16.c -o energy_duel
./energy_duel (on Linux/Mac)
energy_duel.exe (on Windows)

2️⃣ Element Duel

gcc example_17.c -o element_duel
./element_duel (on Linux/Mac)
element_duel.exe (on Windows)

📋 Steps

  1. Clone the repository
  2. Navigate to the project directory
  3. Compile the C file using gcc
  4. Run the executable
  5. Follow the in-game instructions to play!
